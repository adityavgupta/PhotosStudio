# PhotosStudio

A modern photos library application for gnome with gtk4 and libadwaita. There are other applications that exist such as shotwell, which functionally serve the same purpose,
however, it is time for a modern experience.

Modern photos libraries in desktops and phones are database based. This will also be like that.

Much like shotwell, the goal is to point the main view grid to make a timeline by pointing to the photos directories you select to be added into the application. No photos live inside the database, only exif data to create the timeline, tags and other features that contributors want to include. I like how photos in general are stored in directories or folders a user chooses. However, I am also a fan of the viewing experience in dedicated photos applications like apple photos and google photos. 

### My main motivation
1) In my recent transition from moving stuff from my mac to my linux computer, I often felt I wanted the nice look and feel of apple photos. Shotwell exists, but I did not like using it. There may be others (or not) who feel the same way. I wanted to provide an alternative.

2) It is also to practice my Image Processing and Digital Signal Processing skills in cpp.
